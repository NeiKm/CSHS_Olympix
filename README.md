# CSHS Olympix

Простой проект для 천안상업고등학교.

Этот сайт изначально создавался с идеей.
Мне кажется, что если у школьников будет место, где есть **ранг, рейтинг и какие-то награды**, то это может реально повысить интерес и мотивацию развиваться.
Когда ты видишь своё место в таблице и свой прогресс это сильно меняет отношение.

Именно это и есть основная идея проекта.

Поэтому я решил начать с малого сделать сайт только для своей школы.
Сейчас проект в разработке, многое ещё не готово.
База данных планируется (PostgreSQL), но пока ещё не реализована полностью.

Если найдутся люди, которым это интересно (дизайн, frontend, backend и т.д.)
и которые готовы помочь с реализацией я готов довести проект до полноценного релиза и поддерживать его дальше.

## Задачи

* 1. Сделать нормальную структуру базы данных (пользователи, очки, рейтинг и т.д.)

* 2. Реализовать регистрацию пользователей

* 3. Сделать систему входа (логин)

* 4. Добавить страницу профиля (свой профиль)

* 5. Добавить просмотр профилей других учеников

* 6. Реализовать лидерборд (сортировка по очкам)

* 7. Подключить базу данных (PostgreSQL) к проекту

* 8. Сделать API для получения данных (users, leaderboard)

* 9. Связать backend с frontend (fetch / отображение данных)

* 10. Сделать нормальный UI (кнопки, формы, анимации)

* 11. Добавить уведомления (ошибки, успешная регистрация и т.д.)

* 12. Адаптивность под телефон

* 13. Продумать систему очков (за что даются, как считаются)

* 14. Добавить награды / достижения 🏆

* 15. Сделать систему уровней или рангов

* 16. Добавить безопасность (хеширование паролей)

* 17. Ограничить доступ (например, только для учеников школы)

* 18. Деплой проекта (выложить в интернет)

* 19. Поддержка и обновление проекта

# CSHS Olympix

천안상업고등학교를 위한 간단한 프로젝트입니다.

이 사이트는 처음부터 하나의 아이디어를 가지고 시작되었습니다.
학생들에게 **랭크, 점수, 그리고 보상 같은 요소**가 있다면 더 흥미를 느끼고 발전하려는 동기가 생길 수 있다고 생각합니다.
자신의 순위와 성장 과정을 눈으로 확인할 수 있다는 건 생각보다 큰 변화를 만들어냅니다.

이것이 이 프로젝트의 핵심 아이디어입니다.

그래서 우선은 작게 시작해서, 제 학교를 위한 사이트를 만들기로 했습니다.
현재 프로젝트는 개발 중이며, 아직 완성되지 않은 부분이 많습니다.
데이터베이스(PostgreSQL)는 계획되어 있지만 아직 완전히 구현되지는 않았습니다.

만약 이 프로젝트에 관심이 있고 (디자인, 프론트엔드, 백엔드 등)
함께 개발에 참여하고 싶은 분이 있다면, 저는 이 프로젝트를 완성까지 발전시키고 계속 유지할 생각입니다.
